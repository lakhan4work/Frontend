URI Encoding Demonstration

Original (incorrect) URL with spaces and special characters:

https://example.com/search?query=HTML & CSS

Correctly encoded version using %20 for spaces:

https://example.com/search?query=HTML%20%26%20CSS

The %20 represents a space, and %26 encodes the ampersand (&). URI encoding ensures URLs are valid and can be interpreted correctly by web browsers.
